# Support

## Getting Help

- **Issues:** Use the issue templates to report bugs, request features, or flag documentation gaps.
- **Discussions:** Ask questions and share ideas in [GitHub Discussions](https://github.com/orgs/clouatre-labs/discussions).
- **Security:** Report vulnerabilities privately per our [Security Policy](SECURITY.md).

## Response Times

This is a volunteer-maintained project. We do our best to respond promptly, but cannot guarantee specific timelines.
