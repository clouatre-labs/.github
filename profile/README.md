## Contributing

Interested in collaborating? Great! Each project has its own contributing guidelines, but generally:

- Open an issue to discuss ideas before starting work
- Follow the existing code style and patterns
- Write tests for new features
- Keep PRs focused and reasonably sized

See individual project repositories for specific guidelines.

## Get in Touch

- Open an issue in the relevant project
- Start a discussion in the project's Discussions tab
- Connect via <https://github.com/clouatre>
